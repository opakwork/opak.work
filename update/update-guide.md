# Update Lichen-markdown

Use the scripts in this folder to update lichen-markdown or restore an earlier version.

## Usage, update

Run the update script with:

```
bash update_lichen.sh
```

This updates the lichen-markdown installation and downloads a corresponding restore script in this folder.

## Usage, restore

If for some reason you are not happy with the new update, you can roll-back the update by running:

```
bash restore_lichen.sh
```

This will rollback to whatever version you had before you ran the update.
